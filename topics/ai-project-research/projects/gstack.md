# GStack — Garry Tan 的 Claude Code Workflow

## 基本資訊
- GitHub: 8,500+ stars
- 作者: Garry Tan (YC President)
- 發布: 2026-03
- 技術棧: Claude Code + Git + Bun v1.0+
- 授權: MIT

## 它解決什麼問題？

把 AI coding 工作流拆成明確的模式（planning, review, shipping, QA），讓一個人能做五個人的工作。

## 核心功能
- 8 個 opinionated workflow skills
- 分離 product review / engineering review / code review / release / browser QA
- 持久化 browser runtime
- 60 天內寫了 60 萬行 production code，35% 是 tests

## 我能 Build 嗎？

### 技術可行性：⭐ (容易)
- 本質上是一組 Claude Code skills / prompts
- 可以直接用、也可以改成自己的版本
- 不需要建基礎設施

### 可能的切入角度
1. **學習並內化他的工作流**：直接裝來用，優化自己的 coding 流程
2. **做自己的 skill pack**：針對自己的技術棧（Python + Streamlit + FastAPI）客製
3. **中文版 + 教學**：做中文社群的 GStack 教學內容

### 學到什麼
- 頂級工程師的 AI-first 工作流
- Claude Code skill 設計模式
- Product thinking in coding workflow
