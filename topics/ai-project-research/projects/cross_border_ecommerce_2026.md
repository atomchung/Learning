# 中國跨境電商選品研究 — Amazon vs Temu (2026)

研究日期：2026-05-10
範圍：Amazon、Temu 兩個平台，重點是中國賣家視角的選品邏輯與經濟模型

---

## TL;DR

- **大盤訊號**：跨境電商從「野蠻生長」進入「精細化運營」。年銷百萬美元級賣家數量四年翻倍，但活躍賣家總數下降 25%。錢往頭部集中。
- **平台分工**：Amazon 走品牌私域路線（高利潤、高門檻、長週期），Temu 走低價走量路線（低門檻、扣點重、靠規模）。新人不要兩邊同時開。
- **2026 爆品共通邏輯**：能拍短影音 + 解決真實痛點 + 低運費體積 + 高情緒價值。不是「便宜的東西」而是「鏡頭友好的剛需」。
- **個人入門最低資金**：Temu 半托管 ≈ 1.5–3 萬人民幣（含 1 萬保證金）；Amazon FBA ≈ 3–10 萬人民幣（看品類與備貨）。
- **健康利潤帶**：Amazon FBA 私域 25–30% 淨利可達；Temu 全托管 10–20%、半托管 25%+。低於 8% 淨利就是危險訊號。

| 智能個護 | 智能美容儀、頭皮按摩、潔面儀 | 66% 北美消費者願為智能個護付溢價 |

2. **鏡頭友好** — 視覺效果要能在 demo 影片中一眼說明
3. **解決真痛點** — 不是裝飾品，是日常會卡住的小麻煩

## 二、怎麼選爆品（方法論）

### 2.1 選品的四個基本判斷維度

不論 Amazon 或 Temu，賣家社群共識：

1. **體積與重量** — 體積小、輕，運費才會友好。FBA 入倉費直接綁體積，Temu 跨境直發更怕重貨。
2. **需求穩定性** — 季節性過強（聖誕燈、情人節）只能打單波。常青品才有複利。
3. **差異化空間** — 純白牌互捲到死。需要有 USP（獨特賣點）才能撐起溢價。
4. **複購潛力** — 重複消費品（美容、寵物、母嬰）能吃 Subscribe & Save、LTV 高。

### 2.2 數據工具（按用途分）

| 工具 | 用途 | 起價 | 適合誰 |
|------|------|------|--------|
| **Helium 10** | Amazon 全鏈路（關鍵字、競品、PPC） | 年費約 $2,280 起 | 中型以上、肯學習 |
| **Jungle Scout** | Amazon 銷量、利潤估算插件 | $49/月起 | 入門、輕量驗證 |
| **賣家精靈** | 中文 UI、選品+運營監控 | 中文新手友好 | 中國新手首選 |
| **Sorftime** | AI Agent、隱賺指數抓黑馬 | 單次分析約 ¥10 | 2026 新興，AI-native |
| **AmzScout / Amazon 官方選品工具** | 600+ 維度演算法選品 | 部分免費 | 補充驗證 |
| **Google Trends** | 趨勢早期訊號 | 免費 | 任何人都該開 |
| **TikTok / 小紅書 / YouTube Shorts** | 用戶端真實訊號 | 免費 | 所有 Temu 賣家必看 |

### 2.3 實戰選品流程（建議 SOP）

```
Step 1  TikTok / YouTube Shorts 抓 demo 短影音爆款 → 蒐集 30–50 個候選品
Step 2  用 Google Trends 看搜尋曲線（要看到至少 6–12 個月穩定上升）
Step 3  在 Amazon BSR / Temu 銷量榜驗證：類目排名 Top 100 內、前 10 競品評論<500
Step 4  用 Helium 10 / 賣家精靈 反查月銷與利潤模型
Step 5  1688 / 阿里巴巴查供應源 → 落地成本 + 物流費 → 算毛利
Step 6  毛利 > 40%、淨利 > 15% 才上架測款
```

關鍵提醒：**不要找最便宜，要找「高需求、可控競爭、利潤安全、內容可放大」的細分品**。盲選爆品是 2024 之前的打法，2026 已經行不通。

## 三、個人賣家怎麼開始

### 3.1 先選戰場：Amazon vs Temu 路線決策

| 比較項 | Amazon FBA | Temu 半托管 |
|--------|-----------|------------|
| 個人賣家門檻 | 中（需企業/個體戶資質、外幣信用卡） | 低（個人身份證可開 1 店） |
| 啟動資金 | ¥3–10 萬 | ¥1.5–3 萬 |
| 學習曲線 | 陡（PPC、A+ 頁、品牌備案） | 平（平台幫你定價、出單） |
| 利潤上限 | 高（25–35% 私域品牌） | 中（10–25% 半托管偏上） |
| 風險點 | 廣告燒錢、滯銷庫存 | 售後扣款、價格被壓 |
| 出單速度 | 慢（3–6 個月才穩定） | 快（半托管 24h 內可開賣） |
| 適合誰 | 想做品牌、有耐心、有資金 | 副業、有海外倉/現貨、想快速試水 |

**決策建議**：
- 沒做過電商 → Temu 半托管試水（門檻低、容錯高）
- 有產品差異化 / 想做長線品牌 → Amazon FBA
- **不要**兩個同時開新手期，注意力會碎掉

### 3.2 路線 A：Amazon FBA 入門（最常見）

**註冊資料準備**：
- 公司商業文件（個體工商戶或公司皆可）
- 法人身份證
- 可國際付款的雙幣信用卡（Visa/Mastercard）
- 外幣收款帳戶（PingPong、連連、WorldFirst）
- 企業郵箱

**銷售計畫選擇**：
- Individual：無月租，每筆 $0.99
- Professional：$39.99/月，有廣告與 Buy Box 權限 → 想認真做就直接選

**啟動流程**：
1. 註冊（1–3 工作日審核）
2. 選品（用上面 SOP）
3. 1688 / 阿里找供應商打樣
4. 設計品牌、Listing、A+ 頁
5. 頭程貨運到 FBA 倉（空運 5–10 天 / 海運 25–40 天）
6. 開廣告（PPC 投放，約佔銷售額 10–20%）
7. 蒐集 review，目標 30 條 4 星以上 → 進入有機排名階段

**新手最容易翻車的地方**：選品太雷同（紅海互捲）、PPC 不會控成本、首批備貨太多卡死現金流。

### 3.3 路線 B：Temu 半托管入門

**入駐資質**（2026 最新）：
- 大陸主體：個人/個體戶/企業皆可
- 個人身份證 → 只能開 1 個店
- 個體戶/企業執照 → 最多分裂 20 個子店
- 身份證有效期 ≥ 1 月、營業執照 ≥ 3 月

**保證金**：
- 全托管：¥1,000
- 半托管：¥10,000（基礎，必須先繳才能上架）

**模式分支**（2026 因應美國高關稅）：
- **Y1 海外倉模式**：賣家備貨到海外倉，平台派單後賣家本地發
- **Y2 跨境直發模式**：免海外倉、走跨境物流

**入駐流程**：
1. 提交資質審核
2. 繳保證金
3. 上傳爆品候選清單（要圖片高質、價格有競爭力）
4. 選 Y1 / Y2 模式（或混合）
5. 維護庫存 → 平台自動完善詳情並比價
6. 接單發貨

**和全托管最大差別**：半托管你決定價格、自己發貨、保留品牌曝光；全托管你只負責供貨，平台全包但利潤被壓死。**個人賣家想長期做就走半托管**。

### 3.4 起步建議：先用 dropshipping 測款

如果連 ¥1.5 萬保證金都不想壓，最低風險路線：

1. **Shopify 獨立站 + CJ Dropshipping / EPROLO** 一件代發
2. 找 3–5 個候選品，每個花 $50–100 投 TikTok / Meta 廣告測點擊率與轉換率
3. 月銷 ≥ 50 單 → 視為驗證通過 → 轉 Amazon FBA 或 Temu 備貨
4. 月銷 < 10 單 → 砍掉換下一個

這個流程是 2026 新手最低成本驗證方法，把「選品」這件事從紙上推演變成市場真實回饋。

## 四、成本和收益

### 4.1 Amazon FBA 完整成本結構（2026）

**單件商品成本拆解**：
```
售價 100%
├─ 商品成本 (COGS):       20–35%
├─ 頭程運費 + 關稅:        5–10%
├─ Amazon 佣金 (Referral): 8–15% (大多數品類 15%)
├─ FBA 配送費:             10–20% (尺寸決定)
├─ 入倉佈局費:             $0.21–0.68/件
├─ 倉儲費 (月):            (體積 × 日均件數 × 月費率)
├─ 廣告費 (PPC):           10–20%
├─ 退貨損失 + 雜費:        3–5%
└─ 淨利:                  15–25% (好品)
```

**2026 費用變動重點**（生效 2026/1/15）：
- 標準尺寸 FBA 配送費平均 +$0.12/件
- 售價 > $50 的標準品平均 +$0.31/件 — **高客單品衝擊最大**
- Aged Inventory 加收費 1/16 起調整

**啟動資金分配（首次）**：
| 項目 | 金額（¥） |
|------|----------|
| 註冊與資質 | 0–500 |
| 首批備貨（500 件 × ¥30 成本） | 15,000 |
| 頭程運費 | 3,000–8,000 |
| 商標 + 品牌備案 | 3,000 |
| Listing 設計 + 攝影 | 2,000–5,000 |
| 啟動廣告（首月） | 5,000–10,000 |
| 工具訂閱（Helium 10 等） | 500–2,000 |
| **合計** | **¥30,000–50,000** |

**利潤標竿**（業界共識）：
- 私域品牌：25–30% 淨利
- 批發/零售套利：10–20%
- < 8% 淨利 = 商業模式不健康，準備換品

### 4.2 Temu 商家成本結構

**全托管模式**：
- 平台抽成 8–12%
- 賣家只賺差價，定價由 Temu 決定
- 單件淨利常見 ¥3–10
- 整體淨利率：低價競爭品 10–20%、差異化高客單品 30–50%

**半托管模式**：
- 平台抽成 ≈ 15%
- 訂單 < $30 有 $2.99 補貼
- 帳期：T+7+1 天結算
- 賣家自負頭程 + 海外倉 + 尾程
- 淨利率：可達 25%+（看品類）

**售後扣款風險（2026 重點）**：
- 質量問題、發貨超時、訂單取消都會扣款
- 真實案例：14 天扣 ¥66,000、單月扣 ¥37,000
- **合規優秀的賣家比同行多 10% 淨利** — 不是運營能力，是合規能力

**Temu 啟動資金**：
| 項目 | 金額（¥） |
|------|----------|
| 保證金（半托管） | 10,000 |
| 首批備貨 | 5,000–15,000 |
| 海外倉首批入庫費（半托管） | 2,000–5,000 |
| 頭程物流 | 3,000–8,000 |
| **合計** | **¥20,000–38,000** |

### 4.3 收益模擬：一個 ¥30 成本商品的兩個劇本

**Amazon FBA（私域品牌）**：
```
售價:           $19.99
COGS:           $4.30  (¥30)
頭程運費:       $0.80
Referral 15%:   $3.00
FBA 配送費:     $4.00
PPC 攤提:       $3.00
其他/退貨:      $0.80
———————————————
淨利:           $4.09  → 約 20.5% 淨利
```

**Temu 半托管**：
```
售價:           $9.99
COGS:           $4.30
海外倉物流:     $1.50
平台 15%:       $1.50
退貨/扣款:      $0.50
———————————————
淨利:           $2.19  → 約 21.9% 淨利
（但客單價只有 Amazon 一半，需要 2 倍出貨量才能等量）
```

**結論**：兩平台淨利率帶接近，差別在客單價與週轉速度。Amazon 慢但客單高、有品牌資產；Temu 快但無品牌、靠週轉。

## 五、給「個人賣家從零開始」的最小行動方案

如果是你自己想下場做（個人、有 ¥3–5 萬預算、副業）：

**第 1 個月**：
- 開 Shopify 試用站 + CJ Dropshipping 帳號（成本 ≈ $50）
- 用 TikTok / 小紅書找 5 個候選品
- 投 $200 測廣告，看哪個有訊號

**第 2 個月**：
- 確認 1–2 個品 → 備貨 200–500 件到中國工廠
- 同步註冊 Temu 半托管（¥10,000 保證金）+ 上架測試
- 觀察 Temu 出單速度、平台扣款情況

**第 3–4 個月**：
- 如果 Temu 跑通 → 加大備貨、開更多 SKU
- 同時註冊 Amazon Professional 帳號（$39.99/月）
- 把 Temu 跑通的品做品牌化、上 Amazon FBA

**第 6 個月關卡**：
- 月淨利 < ¥10,000 → 砍掉重來、換品類
- 月淨利 ¥10,000–30,000 → 副業階段穩定，保持
- 月淨利 > ¥30,000 → 全職、再投資擴 SKU
