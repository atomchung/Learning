# Coding Agent 比較：Codex vs Antigravity vs Claude Code

## 摘要
從產品經理角度比較三大 AI 編程工具的架構差異：Codex 雲端非同步、Antigravity Agent-first IDE、Claude Code 終端原生。關鍵差異在執行環境隔離、agent 編排、harness 架構。

## 關鍵問題
- 三者的產品定位差異？
- 執行環境怎麼隔離？
- harness 架構為什麼比模型本身更重要？

## 學到的重點
1. Harness > Model：包裝層（工具、權限、上下文管理）比底層模型更決定品質。2. 三種路線：雲端沙箱 vs 本地IDE vs 終端CLI 各有取捨。3. Agent 編排是下一個競爭焦點。

## 延伸方向
深入研究 agent 編排框架;比較 MCP 協議與其他工具協議;研究 AI coding 對開發流程的改變
