# Andrew Ng〈Open Platforms Beat Power Plays〉

2026-06 Andrew Ng 在 X / The Batch 發的一篇**純策略/賽局向**的文章（零技術內容，不碰模型架構或實作）。FB 上看到的中文版是「Agentic 翻」——用 AI agent 把英文原文翻成中文,內容忠實,不是寓言改寫。**這是真事**。

來源：[Andrew Ng on X](https://x.com/AndrewYNg/status/2068039709126017356)

## 背景（兩件真實事件）

- **Anthropic 出 Claude Fable 5**：在 Mythos 模型上加 guardrails,其中夾帶「不准拿來打造競爭性 LLM」的條款,而且一開始是**隱形的**。開發者反彈後 Anthropic 道歉、說會改。
- **美國政府**：商務部出口管制,限制前沿模型輸出給外國民間用戶。

Ng 把兩件擺一起,稱這是「一旦看見就再也無法當沒看見」(hard to unsee)的時刻——賽局轉向的訊號。

## 論證骨架（三拍）

**1. 點出矛盾**
整個 AI 革命是開放點燃的：Transformers 論文是 Google Brain（Ng 老東家）免費公開的。十年來人人可自由用 LLM 技術。領頭公司現在反過來關門,很諷刺。

**2. 拆穿話術**
「防駭客、防生物武器」這種限制合理；但「不准拿我的模型做競爭產品」是夾帶私貨。類比：等於 Google 禁止你用它的搜尋去做競爭性搜尋引擎——這不是安全,是赤裸的權力操弄(power play)。

**3. 核心預測（標題那句）**
封鎖會適得其反,反而加速對手建替代品。真實前例：
- 美國限制中國買晶片 → 加速中國自製晶片
- 中國威脅斷美國稀土 → 加速美國找替代

所以管制前沿模型,只會把所有人（其他國家、開源社群）推去投資「不會被任何人切斷」的開放替代品。**開放平台最後贏過權力操弄。**

## 一句話總結

想用控制存取權來鞏固領先,結果只會加速對手建一個你管不到的開放替代品。

## 跟既有判斷的咬合

- **讀信號不讀數字** (`topics/ai-industry-reading/cards/read-signals-not-surface-numbers.md`)：Ng 不糾結 Fable 5 跑分,讀的是「領頭羊開始關門」這個動作背後的賽局轉向。這篇是該卡的好實例,值得之後升級成卡片時連過去。
- **開源商品化計時器**：封鎖＝幫對手按下計時器加速鍵（接 DeepSeek 那條判斷）。
- **誰掌控存取軌道＝隱藏風險**：跟「免費 API 三種真相」「身份是收費站」同母題,放大到地緣／前沿模型層級。

## 可質疑處

- Ng 是開源/開放的長期既得利益倡議者（DeepLearning.AI、AI Fund 都吃開放生態紅利）=利益相關方,要打折看。
- 「封鎖必反加速對手」用晶片/稀土類比 AI 模型,但模型的複製成本曲線、資料/算力門檻跟實體資源不同,類比未必完全成立——前沿模型的「替代品」要追上頂尖閉源仍有差距,不像稀土那樣替代後等價。
