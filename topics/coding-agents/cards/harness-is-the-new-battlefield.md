# Harness 會是 AI 工具下一階段的競爭主戰場

**一句話**：模型能力趨同後，差異化會從「誰的模型更強」轉移到「誰的 harness 更好」；工程資源、使用者體驗、生態整合等 harness 層決策，將決定贏家。

## 支持這個預測的訊號
1. **Benchmark 趨同**：GPT-5、Claude 4、Gemini 3 在多數 benchmark 差距 < 5%
2. **Harness 被公開討論**：OpenAI、Anthropic 都發表 harness engineering 文章，表示這是戰略資產
3. **同模型不同表現**：Opus 在不同 harness 體感差距大（見 [model-capability-capped-by-harness](./model-capability-capped-by-harness.md)）
4. **新進者從 harness 切入**：Antigravity 用 Harness 差異化，不主打模型

## 反對意見
- 下一代模型可能出現能力斷層（GPT-6？），讓 harness 再次失色
- Harness 優勢可能被快速複製（沒有護城河）
- 終端用戶不懂 harness，市場不會照這邏輯定價

## 如果對，代表什麼
- **投資方向**：應該買 harness-first 公司而非 model-first
- **人才策略**：context engineering、agent orchestration 人才溢價
- **開發者技能**：學會評估 harness 比追模型榜單重要
- **使用者策略**：綁一個 harness 深用，不要頻繁換
